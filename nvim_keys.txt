Neovim key notes
================

Environment
- This config is used on Windows
- Active config path: C:\Users\A\AppData\Local\nvim
- GUI-specific behavior may differ between terminal Neovim and Neovide

General
- Leader key: Space
- Clipboard: uses Windows system clipboard by default via unnamedplus

Window / split movement
- Ctrl+h : move to left split
- Ctrl+j : move to lower split
- Ctrl+k : move to upper split
- Ctrl+l : move to right split

Window resize
- Ctrl+Up    : increase split height
- Ctrl+Down  : decrease split height
- Ctrl+Left  : decrease split width
- Ctrl+Right : increase split width

Buffers
- Shift+h : previous buffer
- Shift+l : next buffer

File tree
- Space e : toggle NvimTree

Scrolling
- Ctrl+d : half-page down and keep cursor centered
- Ctrl+u : half-page up and keep cursor centered

Build / run
- F5 : save current C/C++ file and compile it with Windows g++
- F6 : run the current file's matching .exe in the dedicated build terminal
- F7 : build current CMake project in build-clangd
- F8 : refresh build-clangd/compile_commands.json with CMake
- F5 / F6 / F7 / F8 all reuse the same bottom build terminal pane
- If the build terminal pane was closed, these keys will recreate it automatically

LSP / diagnostics
- K       : hover documentation
- Space k : show diagnostic float
- gd      : go to definition
- gD      : go to declaration
- gi      : go to implementation
- go      : go to type definition
- gr      : list references
- gs      : signature help
- [d      : previous diagnostic
- ]d      : next diagnostic
- F2      : rename symbol
- F4      : code action
- :LspRestart : restart current buffer LSP client

C/C++ notes
- clangd is configured explicitly for the Windows LLVM install
- compile_commands.json is expected in build-clangd
- the current build flow uses PowerShell and Windows executable paths

Completion
- Ctrl+Space : trigger completion
- Ctrl+j     : next completion item
- Ctrl+k     : previous completion item
- Enter      : confirm selected completion item
- Tab        : next completion item when menu is visible
- Shift+Tab  : previous completion item when menu is visible
- Ctrl+b     : scroll docs up
- Ctrl+f     : scroll docs down
- Ctrl+e     : cancel completion

Terminal inside Neovim
- Ctrl+h : move to left split from terminal mode
- Ctrl+j : move to lower split from terminal mode
- Ctrl+k : move to upper split from terminal mode
- Ctrl+l : move to right split from terminal mode
- Ctrl+\ then Ctrl+n : leave terminal input mode and return to normal mode

Split / panel commands
- :sp or Ctrl+w s : split horizontally
- :vs or Ctrl+w v : split vertically
- :close or Ctrl+w c : close current split / panel
- :only or Ctrl+w o : keep only the current split / panel

Editing helpers
- Q : disabled
- Visual mode Alt+j / Alt+k : move selected lines down / up
- Visual mode p : paste without overwriting the unnamed register
- Visual block J / K / Alt+j / Alt+k : move selected block
